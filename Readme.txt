Please compile the files in the following order:

1) mylang.lex file
2) mylang.yy file
3) lex.yy.c and mylang.tab.cc file together using g++

Before running the output file, open preferences of the terminal window, click on your profile, then open the compatibility tab, and change the value of Ambiguous-width characters to Wide.

Now, proceed to running the output file.
The instructions will be visible for 10s, after which you can play the game.

We have made 3 (.h) files that you can include (one at a time) in the mylang.yy file for three different games.
(The main one is the mylang (3 obj).h file.)
